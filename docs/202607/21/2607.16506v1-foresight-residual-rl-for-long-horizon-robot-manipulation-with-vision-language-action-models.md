## 摘要
视觉-语言-动作（VLA）策略提供了强大的通用操作先验，但在高精度、多接触的装配任务中常因长时域信用分配和子任务耦合而失败：对于当前技能几何上成功的状态可能对后续技能脆弱。我们展示了这种失败模式在冻结VLA基策略上的残差强化学习（RL）中：恒定的稀疏成功奖励单独改善每个子任务，但在技能链式执行时收益甚微甚至无增益，因为终端状态质量不受控。我们提出前瞻残差强化学习，通过用离线估计的前瞻价值（基于当前子任务终端状态的未来子任务成功概率）增强每个子任务的稀疏成功奖励来优化交接质量。具体而言，我们（i）从基策略终端状态图像中训练视觉前瞻预测器，并使用下游展开统计标签，以及（ii）通过反向前瞻归纳训练残差策略，将预测器输出作为奖励乘数。在Isaac Gym中的三相扳手螺母拧紧装配任务（抓取、移动插入、旋转）上，我们的方法实现了85.6%的全任务成功率，优于标准子任务残差RL（54.5%）和VLA基线，同时保持各子任务成功率不变。这些结果强调，提升长时域性能需要塑造每个子任务产生哪些成功状态，而不仅仅是是否成功。

## Abstract
Vision-Language-Action (VLA) policies offer strong general-purpose manipulation priors, but often fail on tight-tolerance, contact-rich assembly due to long-horizon credit assignment and subtask coupling: a state that is geometrically successful for the current skill can be brittle for downstream skills. We show this failure mode in residual reinforcement learning (RL) over a frozen VLA base policy: constant sparse success rewards improve each subtask in isolation yet yield little or no gain when skills are chained, because terminal state quality is uncontrolled. We propose Foresight Residual RL, which optimizes handoff quality by augmenting each subtask's sparse success reward with an offline-estimated foresight value -- the probability of future subtask success conditioned on the terminal state of the current subtask. Concretely, we (i) train a visual foresight predictor from images of terminal states of the base policy, labeled using downstream rollout statistics, and (ii) train residual policies via backward foresight induction, using the predictor output as a reward multiplier. On a three-phase wrench-based nut-tightening assembly task in Isaac Gym (grasp, move-insert, rotate), our method achieves 85.6% full-task success, outperforming standard subtask residual RL (54.5%) and VLA baselines, while leaving per-subtask success unchanged. These results highlight that improving long-horizon performance requires shaping which successful states are produced at each sub-task, not only whether success occurs.

- **核心发现**：提升长时域性能的关键在于**塑造子任务终止状态的质量**，而不仅是达成子任务成功。

## 2. 方法论：核心思想、关键技术细节

### 核心思想
提出**Foresight Residual RL**，通过离线估计当前子任务终止状态对未来子任务成功概率（称为**前瞻价值函数**），并将其作为奖励乘数来训练残差策略，从而优化交接状态质量。

### 关键技术细节
1. **阶段划分与残差策略**：将长任务分解为K个顺序子任务，每个子任务有基策略π₀_k 和残差策略π_res_k，最终动作a_t = π₀_k(o_t) + α·π_res_k(o_t)。
2. **前瞻价值函数定义**：对于阶段k，前瞻价值V_{k+1:K}(s) = 从状态s开始，执行后续政策π_{k+1:K}所获得的最终阶段成功折扣回报期望。
3. **局部前瞻价值函数**：将子任务k的稀疏奖励β_k(s_{t+1})乘以V_{k+1:K}(s_{t+1})，得到新学习目标。
4. **反向前瞻归纳训练**：从最后一个阶段开始向前训练，因为训练阶段k需要已固定好的后续阶段政策。
5. **离线前瞻预测器**：
   - 收集：从基策略π₀_k中采集N个成功终止状态。
   - 标注：对每个终止状态，执行K_rep次下游策略π_{k+1}的展开，记录成功次数κ_i，得到标签y_i = κ_i / K_rep。
   - 训练：使用二项式负对数似然损失训练一个视觉预测器p_ϕ(o) → [0,1]。
6. **奖励设计**：训练阶段k时，每一步的奖励为r_t = β_k(s_{t+1}) · p_ϕ(o_{t+1})，仅此一个奖励信号。

## 6. 主要结论与发现

- 标准残差RL提升各子任务独立成功率（如Insert从45.7%→92.2%），但全任务成功率仅54.5%，与端到端π0（54.9%）相当，说明独立优化无助于组合性能。
- 本文方法达到全任务成功率85.6%，比标准残差RL提升31.1%，且各子任务独立成功率变化不大（甚至略有下降），证明优化的是终端状态质量而非成功本身。
- 前瞻预测器不仅提升成功率，还逆转了标准残差RL导致终端状态质量下降的趋势（Grasp终端预测分数从0.32回升到0.73）。
- 视觉前瞻预测器在Grasp→Insert边界准确率86.2%（基线73.9%），Insert→Rotate边界93.9%（基线90.4%）。

## 7. 优点

- **理论优势**：将子任务终止状态质量直接与下游成功概率挂钩，比分布匹配方法更直接，比联合训练更可行。
- **实践高效**：离线收集+学习预测器，避免在线蒙特卡洛展开的高方差和高计算成本；预测器适用于残差策略小幅度偏移。
- **与VLA框架兼容**：使用冻结的VLA基策略，只需训练轻量残差策略和预测器，易于集成。
- **实验设计清晰**：通过对比独立成功率与链式成功率的巨大差距，清晰揭示了问题的本质；终端状态预测分数分析提供了机制解释。

## 8. 不足与局限

- **仿真环境局限**：全部在Isaac Gym中进行，未涉及真实机器人实验。作者指出主要挑战在于残差策略的在线RL成本，而非接触动力学差异。
- **单一任务**：只验证了扳手拧螺母一个任务，且阶段划分是已知的。对更多长时域任务（如通用组装、工具使用）的泛化性未知。
- **一阶近似**：使用下一子任务成功率近似完整前瞻价值，当后续阶段也存在强依赖时可能不够精确（作者指出可扩展为多步滚动）。
