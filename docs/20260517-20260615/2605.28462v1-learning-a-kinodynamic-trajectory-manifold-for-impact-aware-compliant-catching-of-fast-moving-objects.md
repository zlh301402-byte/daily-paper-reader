## 摘要
快速捕获自由飞行物体因反应时间短、冲击不确定性和动力学约束而困难。我们在仿真中使用强化学习收集成功的捕获轨迹，并学习一个低维的动力学轨迹流形。运行时，估计的物体初始状态直接映射到参考捕获轨迹，无需在线非线性优化。在接触附近采用柔顺控制跟踪轨迹，以提高冲击吸收和捕获稳定性。

## Abstract
Fast catching of free-flying objects is difficult because of short reaction time, impact uncertainty, and kinodynamic constraints. We use reinforcement learning in simulation to collect successful catching trajectories and learn a low-dimensional kinodynamic trajectory manifold. At run time, the estimated object initial state is mapped directly to a reference catching trajectory without online nonlinear optimization. The trajectory is tracked with compliant control near contact for improved impact absorption and capture stability.

- **研究动机**：快速移动的物体（如自由飞行的球体）的捕获任务在仓库自动化、空中快递、服务机器人及机器人体育中具有重要应用价值。然而，由于反应时间短、冲击的强烈不确定性以及机器人自身的动力学与运动学约束，实现可靠的快速捕获极具挑战性。

## 论文提出的方法论

- **核心思想**：构建“离线到在线”框架。离线阶段通过强化学习（RL）在仿真环境中探索各类初始状态下成功的捕获轨迹，并学习一个低维的**动力学轨迹流形**（kinodynamic trajectory manifold）；在线阶段，仅需通过感知系统估计物体的初始状态，直接映射到流形上的一个参考轨迹，无需进行耗时的非线性优化。随后，机器人通过近接触区的**柔顺控制**跟踪该轨迹，以实现冲击吸收和稳定捕获。

- **关键技术细节**：
  - **离线数据生成**：将问题建模为有限时域马可夫决策过程。策略输入为物体状态、机器人状态以及相对拦截几何信息（如相对位置、速度、手‑物体对齐）。RL采用三阶段奖励函数：
    - 阶段1（预捕获跟踪）：奖励近距离、速度对齐和前进进度。
    - 阶段2（软接触与缓冲）：在物体进入柔软区（距离<0.3m）后，奖励速度匹配、柔顺跟随，并惩罚峰值接触力。
    - 阶段3（捕获后稳定）：奖励手指包络和残余运动最小化。
    - 此外，包含惩罚项用于避免骤变力冲击、碰撞、关节限位及掉落。
  - **轨迹流形学习**：将RL成功轨迹按拦截时刻对齐并重采样至固定长度H。轨迹表示为14H维向量。成功轨迹构成条件流形M_{xo,0}。通过条件自编码器（Encoder Eψ + Decoder Gφ）将轨迹压缩至低维潜在空间z。训练损失包含：
    - 重构误差L_rec
    - 时间平滑性L_smooth
    - 动力学可行性L_dyn
    - 测地线正则化L_geo：保持流形距离与轨迹空间距离一致
    - 切空间正则化L_tan：鼓励局部线性近似
  - **在线生成**：学得一个条件‑潜在映射Mη: xo,0 → z，使解码器直接输出参考轨迹。也可进一步在潜在空间进行少量梯度下降（利用Riemannian梯度）以微调轨迹。
  - **执行**：生成的参考轨迹由低层柔顺控制器跟踪，在接近接触时切换到阻抗控制以吸收冲击。

- **公式与算法流程**（文字说明）：
  - 目标函数：min J = w1 J_track + w2 J_impact + w3 J_grasp + w4 J_effort，满足动力学约束和几何避碰。
  - 流形参数化：ξ = Gφ(z, xo,0)。
  - 训练损失函数：L_manifold = L_rec + λ1 L_smooth + λ2 L_dyn + λ3 L_geo + λ4 L_tan。
  - 在线生成：ξ_ref = Gφ(Mη(xo,0), xo,0)，或通过潜在空间优化：z* = arg min ℓ(Gφ(z,xo,0), xo,0) + β C_dyn(Gφ(z,xo,0))。
